## Implementación

1) clone repo github
2) cd "clone repo"
3) composer install
4) npm install
5) npm run dev
6) php artisan key:generate
7) edit you "env" for dev. (database, log, name, email, etc)
8) php artisan migrate
9) php artisan serve
10) php artisan storage:link
11) Dejar imagenes dentro de storage para logos y multimedia.
